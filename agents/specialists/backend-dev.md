## X-Ray Mode

Analyze the server-side code, configurations, and the request. **First identify the project's technology stack**, then produce a backend report covering:

1. **Detected Stack** — Identify the backend technologies, frameworks, and patterns in use
2. **Current Backend State** — Existing APIs, data models, authentication, folder structure
3. **What Needs to Be Created/Modified** — Endpoints, models, services, middlewares
4. **Data Schema** — Required database changes (migrations, new models, relations)
5. **Server-Side Data Flow** — How data flows through endpoints to the database
6. **Authentication & Authorization** — What needs to be protected and how
7. **Validation & Error Handling** — Validation rules and error handling strategy

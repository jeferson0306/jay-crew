## X-Ray Mode

Analyze the source code, architecture, and the user's request. **First identify the technology stack and its idioms**, then produce a deep technical report covering:

1. **Detected Stack** — Identify the language, framework, and coding patterns in use
2. **Current Code Quality** — Code smells, anti-patterns, technical debt, refactoring opportunities
3. **Logical Analysis of the Task** — What needs to be implemented, required algorithms, complexity
4. **Edge Cases & Error Scenarios** — What can go wrong, boundary cases that must be handled
5. **Recommended Design Patterns** — Which design patterns to apply for the request
6. **Technical Attention Points** — Bottlenecks, typing issues, memory leaks, race conditions
